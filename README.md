# 🎓 Gerador de Certificados

Uma aplicação full-stack para geração automatizada de certificados em formato PDF. O projeto conta com um front-end moderno em **Next.js** e uma API de geração de documentos construída em **Python (Flask)**.

---

## 🛠️ Tecnologias Utilizadas

### **Front-end**
* **Next.js** (React / TypeScript)
* **Tailwind CSS** (Estilização)

### **Back-end**
* **Python 3.12**
* **Flask** & **Flask-CORS**
* **ReportLab** (Geração e manipulação de arquivos PDF)

### **Infraestrutura**
* **Docker** & **Docker Compose**

---
